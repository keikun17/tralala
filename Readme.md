# Tralala

Mainly for speeding up Rails tests, A gem that exposes and marks the calls that
touches the database.

## Usage

1. Put `gem 'tralala'` in your `Gemfile` under `:group => 'test'`.
2. Run Bundle.
3. Run tests.

5. Profit.
